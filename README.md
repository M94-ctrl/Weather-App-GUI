# Weather-App-GUI
A Python Tkinter GUI weather app using Open-Meteo API (Current, 7-Day, and 30-Day reports).

# Weather App GUI (Python + Tkinter)

A simple weather data application using the **Open-Meteo API**.  
Supports:
- Current weather (shown in GUI console)
- Last 7 days (Excel export)
- Last 30 days (Excel export)

---

### Tech Stack
- Python 3.9+
- Tkinter (for GUI)
- Requests (API calls)
- Pandas (Excel export)
- Open-Meteo API

---

### Run the App
```bash
python WeatherGUI.py
